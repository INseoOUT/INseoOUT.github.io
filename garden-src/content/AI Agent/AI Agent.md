# AI Agent
LLM이 스스로 판단하고 도구를 사용해 목표를 달성하게 만드는 시스템을 공부하는 파트. 만들면서 배운 것을 여기에 쌓는다.

## 학습 지도
### 기초
- [[Agent Loop]] — 에이전트가 생각하고 행동하는 기본 사이클
- [[Tool Use]] — 도구 호출, 함수 콜링
- [[Model Context Protocol|MCP]] — 도구를 표준으로 연결하는 프로토콜
### 구조
- [[Multi Agent System]] — 여러 에이전트를 엮는 법
- [[LangGraph]] — 그래프 기반 에이전트 프레임워크
- [[Hermes]] — 뜯어보며 배우는 중인 에이전트 프레임워크
### 실전에서 배운 것
- fairy — 멀티 에이전트 여행 플래너, [[LangGraph]] Subgraph 라우팅
- slidesmith — [[Model Context Protocol|MCP]] 서버를 내장한 발표자료 도구
- agent-lab — 에이전트 학습·실험 모노레포
